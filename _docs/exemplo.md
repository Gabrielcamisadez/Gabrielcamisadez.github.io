Este é um arquivo de exemplo para demonstrar como seus documentos `.md` serão exibidos.

## Estrutura

Você pode usar a sintaxe normal do Markdown para criar seus documentos.

- Títulos
- Listas
- **Negrito** e *itálico*
- `código`

### Como Funciona

1.  **Adicione um arquivo `.md`** a esta pasta (`_docs`).
2.  **Use o cabeçalho** (front matter) como neste arquivo, definindo pelo menos `title` e `layout: doc`.
3.  O site irá gerar a página automaticamente.
